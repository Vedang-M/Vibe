üîßTroubleshooting

Problem Solution
"fatal: not a git repository" cd /Users/vedangmendhurwar/Downloads/Vibe then git init
"Authentication failed" Use Personal Access Token instead of password. See "Authentication Methods" section above.
Site not showing after enabling Pages Wait 2-3 minutes. Check that index.html exists and Pages is set to "root" folder.
Changes not showing on live site Run git add . && git commit -m "update" && git push origin main
"refused to merge unrelated histories" git pull origin main --allow-unrelated-histories
